Architecture

Overview

Archiview is an autonomous AI-powered code auditing tool that analyzes public GitHub repositories using a 3-pass LLM pipeline (structural analysis, deep code review, aggregation). It features a Node.js/Express backend with modular generators for ASCII art, p5.js scenes, and audio, alongside a rich HTML-based frontend for report visualization.

Express.js (Node.js) JavaScript Monolithic with modular subsystems (director-orchestrated agents, generators, UI components) None (zero-dependency, runs directly with Node.js)
