Architecture

Overview

Hermes Agent is a self-improving, multi-modal AI agent runtime by Nous Research that supports numerous LLM providers and platform integrations via a polyglot monorepo. It organizes a Python agent core with autonomous skill creation and memory loops alongside React-based terminal and web interfaces, plus Node.js messaging bridges. The codebase is heavily plugin-oriented, with separate directories for memory backends, observability, creative skills, and deployment environments.
