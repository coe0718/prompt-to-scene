Verdict
This codebase is structurally ambitious but fundamentally unhealthy for production due to missing CI/CD, unsandboxed dynamic execution, and incomplete security boundaries; the single most important fix is to implement automated testing and sandboxed plugin isolation before any further scaling.

Architecture

Overview

Hermes Agent is a self-improving, multi-platform AI agent built by Nous Research around a Python core with a plugin-based extension system. It exposes functionality through multiple interfaces—including a React-based TUI, a Vite web dashboard, and messaging platform bridges—while organizing capabilities into plugins (memory, observability, integrations) and autonomous skills (creative, research, MLOps). The runtime supports scheduled cron jobs, subagent delegation, and a closed learning loop that refines skills from experience.

Custom Python agent runtime with React/TypeScript frontends Python Plugin-based modular agent architecture with multi-interface gateways setuptools & npm
