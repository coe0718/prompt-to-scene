Architecture

Overview

Hermes Agent is a self-improving AI agent built by Nous Research that features a closed learning loop, autonomous skill creation, and multi-platform deployment. It is organized as a plugin-based monorepo with a Python core, React-based terminal and web UIs, and bridges to messaging platforms like Telegram, WhatsApp, and Discord. The architecture supports scheduled automations, isolated subagents, and pluggable memory backends.

Custom Python agent framework Python Plugin-based monorepo setuptools, npm, Docker Compose
