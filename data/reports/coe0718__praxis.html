Overview

Praxis is a Rust framework for self-hosted autonomous AI agents implementing an OODA (orient-decide-act-reflect-sleep) execution loop with persistent SQLite state, tiered memory management (hot/cold/operational), and safety-first tool governance. It provides a modular runtime with cryptographic identity management, provider-agnostic LLM routing with failover, and optional TUI/web adapters.
