How It Works
Three passes. One verdict.
01
🔭

Structural Analysis

Reads README, config files, directory tree, build system. Maps architecture, framework, and language before touching any source code.

model: Kimi K2.6
02
🔬

Deep Code Review

Every source file chunked and analyzed for code quality, security vulnerabilities, maintainability, and anti-patterns. One file at a time.

model: MiniMax M2.7
03
🧠

Aggregation

All findings merged into prioritized scores across 5 axes. Generates the final report with radar chart, embeddable badge, and optional fix PR.

model: Kimi K2.6
Capabilities
Everything to understand a codebase
📊

5-Axis Scoring

Architecture, Code Quality, Security, Documentation, Maintainability — each scored 0-100 with auto-generated SVG radar chart.

🔍

Prioritized Findings

CRITICAL / WARNING / INFO with exact file paths, line numbers, and concrete fix suggestions you can act on immediately.

⚡

Live Streaming

Watch each pass execute in real time via SSE — structural scan, per-file deep analysis, final aggregation. No blind waiting.

🔧

Smart PR Generation

3-layer safety gates: upstream filter → selector → post-fixer validation. Dry-run only — you decide when to publish.

🏷️

SVG Score Badge

Embeddable badge for any README. Color-coded by overall score, links back to the full report. Show your quality.

📦

Standalone Reports

Self-contained HTML with all CSS inlined. Download, print as PDF, share a link. Works offline, forever.

📈

Historical Trends

Re-audit any repo to track score changes over time. See if your architecture, security, and code quality are improving or slipping.

🏆

Famous Benchmarks

Pre-loaded audits of React, Express, Prettier, and more. Compare your project's scores against industry-leading open-source codebases.
