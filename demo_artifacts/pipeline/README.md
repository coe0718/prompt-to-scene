## The Engineering Story

Behind both artifacts is a pipeline that broke twice before it worked. Two bugs in the JSON-extraction layer caused every LLM response to fail silently:

1. **Double-parse cascade** — returning parsed objects instead of strings, causing downstream crash-to-string coercion
2. **Truncation handling** — O(n²) progressive truncation removed from the wrong direction, making incomplete JSON worse

Both were fixed at the root cause and verified end-to-end. See `success/README.md` for the full diagnosis.
